Agentic Workflow & Architecture

A deep dive into how our multi-agent system orchestrates the learning experience. We utilize advanced LLM orchestration to provide a seamless, adaptive tutoring session.

Context-Aware Integration

LearnFlow is deeply integrated. You can use the AI Assistant directly from whatever content you are currently viewing. If you are reading a chapter on Physics, clicking the AI will automatically load the AI with the exact context of your current Physics lesson, instantly answering your questions without you having to explain where you are or what you are studying.

1

The Profiler Agent

When a user initiates a session, the Profiler Agent retrieves the user's historical data, learning style preferences, and past difficulties. It constructs a dynamic system prompt that sets the tone, vocabulary level, and analogy domain for the session.

2

The Content Generator

The core tutor. It takes the topic and the profile context to generate the actual instructional material. It is instructed to output content in structured chunks rather than massive walls of text, preventing cognitive overload.

3

The Evaluator Agent

Runs continuously in the background. It analyzes the user's responses to questions or their requests for clarification. If it detects high cognitive load (e.g., user asking "I don't get it" multiple times), it signals the Content Generator to drop down a complexity level.

4

The Quiz Master Agent

Triggered when a sub-topic is completed. It dynamically formulates multiple-choice or short-answer questions. It validates the user's answer and provides highly specific feedback on exactly why an answer was wrong, rather than just marking it incorrect.
