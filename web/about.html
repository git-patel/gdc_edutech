About the Project

LearnFlow is an innovative EdTech application designed to revolutionize how individuals approach learning new and complex topics through the power of Agentic AI.

The Challenge

"Create an intelligent assistant that helps users learn new concepts effectively. The system should personalize content and adapt to user pace and understanding."

We tackled this challenge by moving beyond simple prompt-response interactions. Standard AI interfaces often overwhelm learners with too much information or fail to verify if the user actually understood the output. Our solution implements a continuous learning loop driven by multiple AI agents working in tandem to profile the user, generate tailored analogies, test comprehension, and dynamically adjust the curriculum.

Tech Stack

  • Frontend App: Flutter (Dart) with Material UI, ensuring a beautiful and responsive cross-platform experience.
  • AI Engine: Powered by Gemini Pro API, leveraging its massive context window and reasoning capabilities to manage multi-agent workflows.
  • Local Storage: User progress, mastery levels, and chat history are cached locally (SQLite / SharedPreferences) for snappy performance.
